# Drum Machine

A Drum Machine app made in ReactJS. This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Getting Started

```
git clone https://github.com/michaelignacio/drum-machine.git
cd drum-machine
npm install
npm start
```

### Prerequisites

- npm
- create-react-app
